# Interview AI

Welcome to **Interview AI**!

> Built by [Saharsh Singh]

This project uses Google GenAI and standard web technologies to provide an automated, AI-driven interview preparation experience. Upload your resume or write a quick self-description alongside a target job description, and our AI constructs a personalized preparation strategy, complete with technical and behavioral questions, and a day-by-day roadmap.

## Project Structure
- **/Backend**: Node.js/Express application handling the API and AI integration.
- **/Frontend**: React application built with Vite for the user interface.

## Getting Started

### Backend
1. Navigate to the `Backend` directory.
2. Install dependencies: `npm install`
3. Run the server: `npm run dev`

### Frontend
1. Navigate to the `Frontend` directory.
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
